# Navi Mumbai Essential Services Web Map

This repository contains a simple Leaflet-based web map showing essential services in Navi Mumbai, including Police Stations, Fire Stations, UHC/UPHC locations, and Government/Municipal Hospitals.
The map uses OpenStreetMap and Satellite basemaps, with options to toggle layers on or off.

This project is intended for basic GIS visualization and can be opened directly in any web browser.
